This is a sample blog post. The CartaG theme supports **markdown**, code blocks, and more.

## Features

- Dark / light mode
- i18n (EN / ID)
- Search
- Responsive layout

## Code

```bash
hugo new site my site
cd my site
git submodule add https://github.com/Coding-Geh/cartag themes/cartag
```

Thanks for trying CartaG.
